## Key Decisions Made

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| GreenNode embedding over AITeamVN | MAP@5: 42.08 vs lower, 8192 context window | AITeamVN/Vietnamese_Embedding, halong_embedding |
| Vi-Qwen2-7B-RAG over Qwen2.5-3B-KAI | Same architecture (drop-in), RAG-optimized | Qwen2.5-3B-KAI (smaller but different arch), PhoGPT-4B |
| GRPO over DPO for fine-tuning | No paired preferences needed, reward = "Cypher executes" | DPO (needs preference pairs) |
| WRRF over simple score fusion | Handles rank-based fusion across heterogeneous signals | Linear score combination, learned fusion |
| 6-type relation ontology | Covers most common Wikipedia relations without explosion | Open-ended extraction, larger ontology |
