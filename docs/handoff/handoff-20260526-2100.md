## Context Summary

The ViQuAD2.0 benchmark evaluation returns 0% context hit rate and 0% EM/F1 because:
1. Chunks stored in Neo4j contain raw wiki markup (`[[links]]`, `{{templates}}`, `<ref>` tags) while eval gold contexts are clean text
2. Evaluation uses strict substring matching (`ctx_norm in rt.lower()`) which fails with any formatting difference
3. System always abstains due to `max_score < 0.1` threshold being too aggressive

## Key Decisions Made

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| Token-overlap matching (threshold 0.5) | Robust to minor formatting differences between gold context and retrieved chunks | Fuzzy string matching, embedding similarity |
| Strip markup after section parsing | `chunk_text_v2()` needs raw `== heading ==` markers for section detection, but stored text should be clean | Strip before chunking (loses section info) |
| Remove `max_score < 0.1` abstain | Vector scores vary by index config; threshold causes false abstains | Lower threshold to 0.01 |
| Reranker on CPU | CUDA OOM when both embedding model and reranker loaded on GPU | Smaller reranker model |

### Key Code: The Edit to Apply to `src/evaluation.py`

Replace lines 601-618:
```python
        # Context hit: check if gold context appears in any retrieved chunk
        ctx_norm = context[:200].lower()
        hit = 1.0 if any(ctx_norm in rt.lower() for rt in retrieved_texts if rt) else 0.0
        hit_rates.append(hit)

        # MRR based on context match
        rr = 0.0
        for rank, rt in enumerate(retrieved_texts, 1):
            if rt and ctx_norm in rt.lower():
                rr = 1.0 / rank
                break
        mrrs.append(rr)

        # Rerank
        reranked = rerank(question, rows, text_key="chunk_text", top_k=top_k_rerank)

        # Generate answer from top reranked chunks
        system_abstains = len(rows) == 0 or max_score < 0.1
```

With:
```python
        # Context hit: token overlap between gold context and retrieved chunks
        ctx_tokens = set(context.lower().split())
        hit = 0.0
        if ctx_tokens:
            for rt in retrieved_texts:
                if not rt:
                    continue
                chunk_tokens = set(rt.lower().split())
                overlap = len(ctx_tokens & chunk_tokens) / len(ctx_tokens)
                if overlap >= 0.5:
                    hit = 1.0
                    break
        hit_rates.append(hit)

        # MRR based on token overlap
        rr = 0.0
        if ctx_tokens:
            for rank, rt in enumerate(retrieved_texts, 1):
                if not rt:
                    continue
                chunk_tokens = set(rt.lower().split())
                overlap = len(ctx_tokens & chunk_tokens) / len(ctx_tokens)
                if overlap >= 0.5:
                    rr = 1.0 / rank
                    break
        mrrs.append(rr)

        # Rerank
        reranked = rerank(question, rows, text_key="chunk_text", top_k=top_k_rerank)

        # Generate answer from top reranked chunks
        system_abstains = len(rows) == 0
```
