## Key Decisions Made

| Decision | Rationale | Alternatives Considered |
|----------|-----------|------------------------|
| Use igraph+leidenalg for communities | Neo4j GDS is paid/enterprise only | networkx (no Leiden), Neo4j GDS |
| Default model: NlpHUST/ner-vietnamese-electra-base | vilm/videberta-base-ner doesn't exist on HF | PhoBERT-NER, XLM-R-NER |
| wRRF fusion with 3 channels | Proven in Blended RAG paper (87% accuracy) | Simple concatenation, learned fusion |
| Parallel trajectories default=1 | Avoids cost multiplication unless explicitly enabled | Always-on (expensive) |
| GreenNode already configured as default embedding | Already in config.py, 1024d, 8K context | Vietnamese_Embedding_v2 (faster but same dim) |
